How to use
1
Upload your file. Drag and drop a CSV or Excel file or click browse files to select one.
2
Select columns. Choose which columns to keep (appear in every output row) and which to pivot (convert to rows).
3
Run conversion. Click Run Conversion & Download to unpivot your data.
4
Download result. The converted file is automatically downloaded as output-data.csv.
